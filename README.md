# TornPaper 

## Demo 


## Usage 
### html
```
<script src="https://cdn.jsdelivr.net/gh/happy358/TornPaper
/tornpaper.min.js"></script>
<script>
    new Tornpaper().run;
</script>
```

### css
```
:root {
    --grungepaper: url();
}
.your-class {
background: var(--grungepaper) center/cover repeat;
filter: url(#filter_tornpaper);
}
```
 
## Options 
```
    new Tornpaper({
        grunge: true
    }).run;
```
